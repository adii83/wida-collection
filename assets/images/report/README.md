# Bukti Screenshot Modul 4

Simpan file screenshot sesuai daftar berikut. Nama file digunakan juga di `docs/modul4_eksperimen.md`:

1. `theme-shared-prefs.png` – layar Pengaturan Tema yang menunjukkan segmented button dan pilihan warna yang tersimpan.
2. `hive-offline.png` – layar Catatan Offline (Hive) saat perangkat offline.
3. `supabase-cloud.png` – layar Catatan Cloud dengan data tersinkron.
4. `multi-device.png` – dua perangkat/login berbeda yang menampilkan data Supabase yang sama.

> Gunakan emulator/perangkat fisik untuk mengambil gambar, lalu simpan file PNG di folder ini agar bisa dikomit bersama laporan.
